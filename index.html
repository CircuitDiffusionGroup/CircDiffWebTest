Features

Why choose Tracer HDL?

Designed with PCB engineers in mind, Tracer brings modern language features to hardware design.

PCB-Centric Design

Tracer is built specifically for PCB design workflows, with native support for common PCB components and constraints.

Modern HDL Syntax

Enjoy clean, expressive syntax with features like type inference, generics, and functional programming constructs.

Fast Compilation

Optimized compiler pipeline delivers fast iteration cycles, even for complex designs.

Multi-Format Output

Generate netlists, BOMs, and documentation in multiple formats from a single source.

Static Verification

Catch design errors early with built-in static analysis and verification tools.

Open Source

Fully open-source with an active community contributing to its development.

Getting Started

Begin your Tracer journey

Follow these steps to set up Tracer HDL and create your first design.

  • 1

    Install Tracer

    Download and install the Tracer compiler for your platform.

    # Install code yet to be made, distributed as a binary ATM
  • 2

    Create a Project

    Create a tracer file.

    $ touch design.t
  • 3

    Write Your Design

    Create your hardware design using Tracer HDL syntax.

    // Example: Simple led connected to a battery
    led(anode=VCC, cathode=R1);
    resistor(a=R1, b=GND);
    bat(p=VCC, n=GND);
  • 4

    Compile & Export

    Compile your design and export to various formats.

    $ tracer -t design.t -o out.cds
